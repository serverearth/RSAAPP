Tentang Aplikasi
Aplikasi Pengamanan File Dokumen ini menggunakan Algoritma RSA. RSA di bidang kriptografi adalah sebuah algoritma pada enkripsi public key.

Generate Kunci
Proses membuat kunci RSA untuk menghasilkan Private dan kunci Public. Anda bisa melakukan proses ini dengan mengikuti proses dibawah ini :
  • Jalankan Aplikasi
  • Buka menu RSA pilih Generate Kunci
  • Form generate kunci akan tampil, dimana anda bisa melakukan proses generate kunci untuk menghasilkan kunci Private dan kunci Public

Penambahan User
Jika User Anda bertambah atau karyawan Anda bertambah, anda bisa menggunakan fitur penambahan user. user dibedakan menjadi dua bagian, user untuk pusat dimana user tersebut adalah user hak sepenuhnya pemegang kunci private, dan selanjutnya user cabang dimana user tersebut adalah user untuk melakukan proses enkripsi.

Distribusi Kunci
Pendistribusian kunci dilakukan secara manual, melalui media penyimpanan seperti falshdisk dan lain sebagainya dengan ketentuan melakukan pendistribusian harus lah aman dan kunci yang akan di distribusi adalah kunci publik.

Enkripsi Dokumen
Untuk melakukan proses enkripsi dokumen, pilih menu RSA dan klik sub menu Enkripsi. Akan muncul sebuah form untuk melakukan proses enkripsi.

Dekripsi Dokumen
Untuk melakukan proses enkripsi dokumen, pilih menu RSA dan klik sub menu Dekripsi. akan muncul sebuah form untuk melakukan proses dekripsi.
